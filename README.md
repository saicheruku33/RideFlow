# RideFlow

RideFlow is a backend cab booking application built using **Java** and **Spring Boot**. The project simulates the core functionality of a ride-hailing platform by allowing users to book rides, manage drivers and cabs, and track trip details through RESTful APIs.

## Features

* User and driver management
* Cab registration and availability management
* Ride booking and trip tracking
* Fare calculation based on trip distance
* RESTful APIs for all major operations
* Database integration using Spring Data JPA and Oracle SQL
* Clean layered architecture with Controller, Service, Repository, and Entity layers

## Tech Stack

* Java
* Spring Boot
* Spring Data JPA
* Hibernate
* Oracle SQL
* Maven


## Getting Started

1. Clone this repository.
2. Configure your Oracle database connection in `application.properties`.
3. Build the project using Maven.
4. Run the Spring Boot application.
5. Use Postman or any API client to test the available endpoints.
